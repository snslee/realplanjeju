# realplanjeju

리얼플랜제주 주식회사 공식 홈페이지 및 고객 문의 시스템.

## 구조

- `index.html` — 메인 포털
- `contact.html` — 고객 문의 폼 (Phase 1)
- `privacy.html` — 개인정보처리방침 (법적 필수)
- `sql/` — Supabase 스키마 DDL
- `assets/` — CSS · JS · 이미지

## 기술 스택

- Frontend: HTML5 + CSS3 + Vanilla JS
- Database: Supabase PostgreSQL (서울 리전)
- Hosting: GitHub Pages (Private 저장소 + 커스텀 도메인)
- Automation: Make (Phase 2+)
