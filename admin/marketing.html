🎨 톤 가이드

채널별 콘텐츠 작성 기준

🏔️ 블A — 여행 (네이버블로그)

말투
친근한 존댓말 (-해요/-이에요) / 반말 금지 / 1인칭 "저는~"
제목 패턴
[지역] + [핵심경험] + [감성키워드]
예) "제주 성산 현지인 맛집, 2박3일 숨은 코스"
핵심 감정
설렘 · 힐링 · 특별함 · 추억 · 로컬 감성
본문 구조
배경(왜 갔나) → 과정(직접 경험) → 감동 포인트 → 실용 팁
⛔ 금지어
최고 · 최저가 · 보장 · 무조건 · 꼭 가야해

🎪 블B — 행사 (네이버블로그)

말투
격식체 (-합니다/-입니다) / B2B 신뢰 우선 / 수치 강조
제목 패턴
[행사종류] + [규모/특징] + [성과]
예) "기업 워크샵 200명, 제주 3일 성공 운영 사례"
핵심 키워드
전문성 · 신뢰 · 실적 · 숫자 · 맞춤 기획
본문 구조
결론(성과) → 기획 배경 → 진행 과정 → 고객 반응
⛔ 금지어
저렴 · 싸게 · 대충 · 막연한 미사여구

📚 블C — 마케팅교육 (티스토리)

말투
친근한 격식체 (-할 수 있어요/-됩니다) / 전문가 포지션 유지
제목 패턴
[숫자] + [구체 방법/결과]
예) "소상공인 블로그 3개월 만에 방문자 1만 만드는 법"
핵심 키워드
구체적 숫자 · 실전 팁 · 실증 사례 · 따라하기 쉬운
본문 구조
문제 공감 → 원인 진단 → 해결책 제시 → 단계별 실행
⛔ 금지어
무조건 · 쉽게 · 누구나 · 100% · 무조건 성공

📐 공통 SEO 원칙

🔤 제목
핵심 키워드 첫 10자 이내 배치 / 25~35자 최적
📝 본문
2,000자 이상 / h2·h3 목차 필수 / 단락 4~6줄
🖼️ 이미지
alt 텍스트 키워드 포함 / 대표이미지 필수
🔗 내부링크
관련 글 2~3개 링크 / 클러스터 구조
⏱️ 발행 시간
평일 오전 9~11시 / 주말 오전 8~10시
🤖 AI 주의
생성형 AI 그대로 복붙 금지 / 직접 경험 필수 추가
